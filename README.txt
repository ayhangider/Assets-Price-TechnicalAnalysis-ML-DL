
Tried to understand which technical analysis variables are more important about crypto currency prices
in short and long intervals.

The CryptoDataCollector class downloads the current prices for ticker. After that, it calculates the 
most popular & useful technical analysis variables for ticker and intervals.

You can try like this:
Crc_avax = CryptoDataCollection(ticker='AVAX-USD', prediction_interval='5m') 

It will give you the price data of AVAX-USD and also t.a. variables. You can try for your 
price prediction projects.
